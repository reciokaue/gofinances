## Getting Started

### Prerequisites

* npm | yarn
* React Native 
* Expo

## Installation

1. Clone the repo
   ```sh
   https://github.com/reciokaue/gofinances.git
   ```
2. Install NPM packages using yarn or npm

   ```js
   yarn
   ```
   ```sh
   npm install
   ```
3. Run the app
    ```sh
    expo start
    ```
